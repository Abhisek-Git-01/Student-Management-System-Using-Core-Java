# 🎓 Student Management System (Java + JDBC)

A **console-based Java application** for managing student records, supporting both **Admin** and **Student** roles. This system uses **Core Java**, **JDBC**, and **MySQL** to allow CRUD operations (Create, Read, Update, Delete) on student data.

---

## 👥 User Roles

### 🛠️ Admin
Admins have full access to manage the system and can:
- ➕ Add new student records
- 📝 Update student details (name, course, marks)
- 🔍 View student details by ID
- ❌ Delete student records
- 📋 View all students (optional enhancement)

### 👨‍🎓 Student
Students can:
- 🔍 View their own profile by ID
- 📝 Request updates (in a real system, this could be moderated)
- 👁️‍🗨️ Check their marks and course info

---

## 🚀 Features
- Role-based access control (basic)
- Full CRUD operations
- Data persistence with JDBC and MySQL
- Input validation for types and values
- Modular architecture (Entity, DAO, Service, UI)
- Console-based UI using `Scanner`

---

## 🛠 Technologies Used
- Java 17+
- JDBC (Java Database Connectivity)
- MySQL
